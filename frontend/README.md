# File Upload Frontend

React + Vite application for staged file uploads.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Features

- Add files to a staging list with Claim ID and Document Code
- Remove files from the staging list
- Upload all staged files to the backend API
- Real-time UI updates using React state management

## Backend Integration

The app is configured to communicate with the backend API at `http://localhost:3000`. Make sure your backend server is running before uploading files.
